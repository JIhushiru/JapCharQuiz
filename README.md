# JapCharQuiz

A Japanese character quiz app for learning hiragana and katakana. Test your knowledge by typing the correct romaji for each character.

## Features

- Hiragana, Katakana, or combined quiz modes
- Score and streak tracking
- High scores saved locally
- Skip and reveal answer options

## Getting Started

```bash
npm install
npm run dev
```

## Tech Stack

- React
- TypeScript
- Vite
- React Router
